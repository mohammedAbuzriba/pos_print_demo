# pos_print_demo

مشروع طباعة فورية على ماكينات POS عبر البلوتوث، مخصص للاستخدام في أنظمة الطلبات والكاشير.

---

## وصف المشروع

يوفر هذا المشروع آلية **الطباعة عبر Bluetooth POS Printer** داخل تطبيق Flutter،  
مع إمكانية ربط الطابعة مرة واحدة فقط ثم استخدامها تلقائيًا عند تنفيذ العمليات  
(مثل قبول الطلب – قيد التجهيز).

يعتمد المشروع على أسلوب مستخدم في أنظمة POS المحمولة، دون الحاجة لتعريف الطابعة كنظام تشغيل.

---

## آلية العمل باختصار

- ربط الطابعة عبر البلوتوث من صفحة الإعدادات (مرة واحدة).
- حفظ بيانات الطابعة (MAC Address + Name).
- الطباعة التلقائية لاحقًا عند تنفيذ الطلبات.
- دعم اللغة العربية (RTL) عبر تحويل الإيصال إلى صورة قبل الطباعة.

---

## المكتبات المستخدمة

- **print_bluetooth_thermal**  
  للتعامل مع البلوتوث (جلب الطابعات المقترنة، الاتصال، إرسال بيانات الطباعة).

- **esc_pos_utils_plus**  
  لتجهيز أوامر الطباعة بصيغة ESC/POS المتوافقة مع أغلب طابعات POS.

- **permission_handler**  
  لإدارة صلاحيات البلوتوث المطلوبة حسب إصدار أندرويد.

- **image**  
  لمعالجة الصور المستخدمة في طباعة النص العربي.

---

## الصلاحيات المطلوبة (Android)

- Android 12+

  - BLUETOOTH_CONNECT
  - BLUETOOTH_SCAN

- Android 11 وما قبله
  - BLUETOOTH
  - BLUETOOTH_ADMIN
  - ACCESS_FINE_LOCATION

> بدون هذه الصلاحيات لن يتمكن التطبيق من الاتصال بالطابعة أو تنفيذ الطباعة.
